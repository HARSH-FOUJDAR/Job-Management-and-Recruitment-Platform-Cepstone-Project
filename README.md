# Job Management and Recruitment Platform

A full-stack MERN application that connects students with recruiters through a modern job portal.

## Live Demo

🔗 

## Features

* User Authentication & Authorization
* Student Dashboard
* Recruiter Dashboard
* Job Posting and Management
* Job Search and Applications
* Application Tracking
* Profile Management
* Responsive Design

## Panels

### Student Panel

* Create and manage profile
* Search jobs
* Apply for jobs
* Track application status

### Recruiter Panel

* Post new jobs
* Manage job listings
* View applicants
* Update application status

## Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* JWT Authentication
* Redux Toolkit
* Tailwind CSS
